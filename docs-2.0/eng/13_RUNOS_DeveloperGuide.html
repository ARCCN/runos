RUNOS Developer Guide¶

Requirements to RUNOS developer¶

  • Linux operating system skills
  • C++17 skills
  • QT skills
  • CMake skills
  • SDN skills
  • knowledge of the OpenFlow specification 1.3

RUNOS Core¶

  • Application Class
  • Loader Class
  • Logger Class

Application¶

Application can have one of the following states:
  • APP_REGISTERED,
  • APP_INITIALIZING,
  • APP_INITIALIZED,
  • APP_STARTED.

Loader¶

Loader states:
  • INITIALIZING,
  • STARTING,
  • RUNNING.
