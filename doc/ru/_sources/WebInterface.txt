Web interface
=============

У Вашего приложения может быть своя html-страница. Для того чтобы добавить ее, реализуйте в классе Вашего приложения метод ::

    std::string page() override { return "my_page.html"; }
Файл ``"my_page.html`` должен быть расположен в дирректории ``web/html``.
Ваше приложение будет отображаться в списке приложений под именем::

        std::string displayedName() override { return "My beautiful application"; }  // web name

Чтобы обратиться к уже существующеу web-интрефейсу, откройте http://127.1.1.0:8000/topology.html

////////////////////

Веб-интерфейс в RuNOS можно понимать как стороннее приложение, опирающееся на rest-интерфейс контроллера
